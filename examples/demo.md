# AI 时代的开发者工作流

## 概述

在 AI 辅助编程的时代，开发者的工作流程正在发生根本性的变革。

### 核心变化

- **代码生成**：从手写到 AI 生成 + 人工审核
- **调试方式**：自然语言描述问题，AI 分析根因
- **文档编写**：AI 理解代码意图，自动生成文档

> 未来的开发者不是写代码的人，而是和 AI 一起思考的人。

### 代码示例

```javascript
async function askAI(prompt) {
  const response = await gemini.generate(prompt);
  return response.text;
}
```

## 总结

1. 拥抱变化，持续学习
2. 专注于高阶思维
3. 让 AI 处理重复工作
